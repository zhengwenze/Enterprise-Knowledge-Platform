# EKP AI Service

Enterprise Knowledge Platform - AI Service

## Project Overview

AI service for the Enterprise Knowledge Platform, providing RAG (Retrieval-Augmented Generation) capabilities including document processing, vector storage, and intelligent Q&A.

## Tech Stack

- Python 3.11
- FastAPI
- LangChain
- Sentence Transformers
- pgvector
- SQLAlchemy

## Quick Start

```bash
pip install -r requirements.txt
python src/main.py
```

## API Documentation

After starting the service, visit http://localhost:8000/docs for Swagger UI documentation.
