# EKP Business Service

Enterprise Knowledge Platform - Business Service

## Project Overview

Business service for the Enterprise Knowledge Platform, handling user management, document metadata, and business logic.

## Tech Stack

- Java 21
- Spring Boot 3.x
- Spring Data JPA
- PostgreSQL
- Maven

## Quick Start

```bash
mvn spring-boot:run
```

## API Documentation

After starting the service, visit http://localhost:8080/swagger-ui.html for API documentation.
